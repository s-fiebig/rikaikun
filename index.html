rikaikun

This is a Rikaikun fork (https://code.google.com/p/rikaikun/) that supports additional languages besides English.

Installation

  1. Grab the newest release under: https://github.com/s-fiebig/rikaikun/releases
  2. Open the extension in Chrome using developer mode.

Supported Languages

  • English: Default language.

  • German: From the German Wadoku project.

  • ...

Licenses
